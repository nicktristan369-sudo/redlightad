# PWA Icons

Generate icons for the Progressive Web App using your logo.

## Required sizes:
- icon-72x72.png
- icon-96x96.png
- icon-128x128.png
- icon-144x144.png
- icon-152x152.png
- icon-192x192.png
- icon-384x384.png
- icon-512x512.png

## How to generate:
1. Use https://realfavicongenerator.net/ or similar tool
2. Upload your logo (ideally square, 512x512 or larger)
3. Generate icons and download
4. Place them in this folder

## Tips:
- Use a simple, recognizable design
- Ensure it works on both light and dark backgrounds
- The 192x192 and 512x512 are the most important
